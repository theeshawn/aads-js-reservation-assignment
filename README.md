# AADS JavaScript Reservation Assignment

## Project Overview
This project is a JavaScript-based reservation system developed as part of the **AADS JavaScript assignment**.  
It demonstrates core JavaScript concepts including variables, functions, conditional logic, arrays, and basic program flow.

The goal of the assignment is to simulate a simple reservation process using JavaScript logic.

---

## Objectives
- Practice JavaScript fundamentals
- Implement logical conditions and control flow
- Work with data structures (arrays & objects)
- Improve problem-solving using JavaScript

---

## Technologies Used
- **JavaScript (ES6)**
- **Git & GitHub** for version control
